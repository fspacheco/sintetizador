Modulator

O modulator será o software utilizado para exemplos, aqui faremos uma introdução ao modulator para melhor entendimento do leitor. O modulator se encontra separado com 6 módulos diferentes, que serão discutidos a seguir.

Audio Node Graph

O Audio Node Graph é o módulo onde ocorre a construção do modulador, podendo ser colocados e ligados diferentes blocos para diferentes efeitos no som, a ligação dos blocos é feita com o botão SHIFT ao clicar em um bloco e em outro em seguida, e para desfazer a ligação, apenas repita o processo com blocos já ligados.

Palette

O módulo Palette é o módulo onde se encontram os blocos a serem colocados no Audio Node Graph, basta clicar em um bloco e ele será adicionado ao Audio Node Graph, cada bloco possui propriedades diferentes e serão trabalhados mais a frente.

Node Parameters

O módulo Node Parameters serve para modificar o funcionamento de um bloco selecionado no Audio Node Graph, mudando seus parâmetros.

Presets

O módulo Presets serve para salvar e reutilizar modificações feitas pelo usuário, ele também possui opções padrão já salvas como exemplos.

Piano Keyboard

É o teclado do sintetizador, utilizado para tocar as notas com as modificações estabelecidas, também é possível mudar as oitavas, o portamento e o arpejo do instrumento nesse módulo.

Graph

O módulo Graph é um módulo exclusivo para visualização de onda, podendo ver a forma de onda uma vez que tocar em uma nota no teclado.
