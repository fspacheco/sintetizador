Sintetizador Sobre

Projeto Sintetizador

Este projeto foi realizado pelos alunos do IFSC Thiago da Cruz e José Victor Sant’Ana do curso técnico de eletrônica, sendo a atividade realizada em função do estágio, tendo como orientador o professor Fernando Pacheco.

O projeto do site foi feito como uma tradução e adaptação ao guia para sintetizadores feito por Beau Sievers, com mudanças visuais, como na diferença entre as imagens criadas para esse projeto.

Para criação do sintetizador no site, foi utilizado um tutorial disponibilizado pelo Mozilla, que traduzimos e colocamos o link da tradução no site, além de uma explicação do processo resumida.
